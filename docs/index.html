Overview

BuddyBot is a fully autonomous, interactive robot designed to assist with daily tasks, communicate with devices, and provide entertainment. Integrating cloud communication, voice recognition, IoT, and automation, BuddyBot adapts to your needs.

Features

Cloud-Based Communication

Real-time communication using Firebase Cloud Messaging for seamless sync.

Voice Recognition

Understand and respond to your commands with Google Speech-to-Text.

Face Recognition

Personalized interaction using OpenCV for face detection.

Gesture Control

Control BuddyBot with simple hand gestures detected by a camera.

IoT Integration

Automate tasks by controlling smart devices like lights and fans.

Emergency Alerts

Real-time alerts for smoke, motion, or gas detection.
